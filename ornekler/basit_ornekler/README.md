# Gümüşdil Kod Örnekleri

Bu klasör, Gümüşdil programlama dilinin özelliklerini gösteren örnek programlar içerir.

## Örnek Listesi

### Temel Seviye
1. **01_selam.tr** - Basit selam programı (girdi/çıktı)
2. **02_vki.tr** - Vücut Kitle İndeksi hesaplayıcı (koşullar)
3. **03_tek_cift.tr** - Tek/çift sayı kontrolü
4. **04_asal_sayilar.tr** - Asal sayı bulucu (fonksiyonlar)
5. **05_fibonacci.tr** - Fibonacci dizisi (özyineleme)

### Orta Seviye
6. **06_carpim_tablosu.tr** - Çarpım tablosu (iç içe döngüler)
7. **07_ogrenci_not.tr** - Öğrenci not sistemi (OOP)
8. **08_metin_ters.tr** - Metin ters çevirme ve palindrom
9. **09_sesli_harf.tr** - Sesli harf sayacı (Türkçe karakter desteği)
10. **10_sezar_sifre.tr** - Sezar şifreleme algoritması

### İleri Seviye
11. **11_hesap_makinesi.tr** - Tam özellikli hesap makinesi (OOP, geçmiş)
12. **bağlı_liste.tr** - Bağlı liste veri yapısı (pointer görselleştirme)

## Öğrenme Yolu

### 1. Başlangıç (01-03)
- Temel girdi/çıktı işlemleri
- Değişkenler ve veri tipleri
- Koşullu ifadeler (eğer/değilse)

### 2. Fonksiyonlar (04-06)
- Fonksiyon tanımlama ve çağırma
- Parametreler ve dönüş değerleri
- Döngüler (döngü)

### 3. Nesne Yönelimli Programlama (07, 11)
- Sınıf tanımlama
- Kurucu (constructor) kullanımı
- Metotlar ve özellikler (öz)

### 4. Veri Yapıları (12)
- Listeler ve sözlükler
- Bağlı liste implementasyonu
- Bellek yönetimi

## Nasıl Çalıştırılır?

### IDE ile (Önerilen)
```bash
python -m src.ide.main
```
Ardından IDE'de dosyayı açıp **F5** tuşuna basın.

### Komut Satırı ile
```bash
gumus.exe examples/01_selam.tr
```

## Önemli Notlar

### Türkçe Syntax Kuralları
- **Anahtar Kelimeler**: `eğer`, `döngü`, `dön`, `değişken`, `sınıf`, `fonksiyon`
- **Boş Değer**: `boş` (null)
- **Boolean**: `doğru`, `yanlış`
- **Özel**: `öz` (self/this)

### Syntax Özellikleri
- Süslü parantezler `{}` zorunludur
- Noktalı virgül `;` opsiyoneldir
- UTF-8 tam desteği (Türkçe karakterler)

### Bellek Görselleştirme
IDE'de sağ panelden **"Gümüşhafıza"** sekmesini açarak:
1. **"Çağrı Kulesi"** - Stack'i gösterir
2. **"Hafıza Odaları"** - Heap'i gösterir
3. **"Zaman Makinesi"** - Adım adım bellek değişimlerini izleyin
